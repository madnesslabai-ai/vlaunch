# Examples Index

Example outputs generated by vLaunch for real products.

## Complete Examples

### `examples/oddsflow-ai/` — AI-Enhanced Pipeline

Full AI-enhanced output for OddsFlow, a sports analytics product. Includes all 8 assets plus manifest and consistency report. Best showcase of the complete pipeline.

**Files:** positioning.md, routing-plan.md, producthunt.md, medium-draft.md, directories.json, affiliate.md, checklist.md, consistency-report.md, launch-manifest.json

### `examples/oddsflow/` — Deterministic Baseline

Deterministic-only output for the same product (no API key required). Shows what the pipeline produces without AI enhancement.

**Files:** positioning.md, medium-draft.md, producthunt.md, directories.json
